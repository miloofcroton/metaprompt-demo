# How to get set up to use this repo

1. install claude code CLI:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

2. sign into claude and ensure you have a pro (starts at $20/month) or max (starts at $100/month) subscription.

3. install prompt tool:

```bash
npx get-shit-done-cc
```

4. run claude code CLI

```bash
claude
```

5. test GSD plugin (inside claude code)

```bash
/gsd:help
```
